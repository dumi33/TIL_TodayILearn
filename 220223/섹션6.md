### 문제 1 - 재귀함수를 이용한 이진수 출력


- <내가 만든 코드>
```py

def dfs(x) :
    if x >0:
        tmp = x//2
        dfs(tmp)
        print(x%2,end = '')
if __name__ == "__main__" :
    n = int(input())
    dfs(n)
 ```
 
 - <강사님이 만드신 코드>
```py
def dfs(x) :
    if x == 0 :
        return 
    else :
        dfs(x//2)
        print(x%2,end = '')
       
        
if __name__ == "__main__" :
    n = int(input())
    dfs(n)
```

### 문제 2 - 이진트리 순회


- **전위순회**
```py

def dfs(x) :
    if x > 7 :
        return 
    else :
        print(x,end=' ')
        dfs(x*2) # 왼쪽 자식
        dfs(x*2+1) # 오른쪽 자식     
        
if __name__ == "__main__" :
    dfs(1)
```

### 문제 3 - 부분집합 구하기 

```py
def dfs(v) :
    if v == n+1 : # 종료지점
        for i in range(1,n+1) :
            if ch[i] == 1 :
                print(i, end = ' ')
        print()
    else :
        ch[v] = 1  # 왼쪽으로 # 사용
        dfs(v+1)
        ch[v] = 0 # 오른쪽으로 # 사용 X 
        dfs(v+1)

        
if __name__ == "__main__" :
    n = int(input())
    ch= [0] *(n+1)
    dfs(1)
```
